# Codex App Quickstart

This workflow is designed for a normal user with the Codex app, not a custom tax stack.

## First Run

1. Open the Codex app in this repo.
2. Make sure Gmail and Google Drive are connected if that is where your tax forms live.
3. Start with:
   `Use $tax-avoidance to gather my 2025 tax documents and assemble a draft return package.`
4. If Codex finds only a portal notice or encrypted attachment, upload the actual PDF.

## What The Skill Should Do

- find likely forms
- build a document inventory
- ask only the missing interview questions
- build the four standard artifacts
- separate confirmed values from candidate values

## What The Skill Should Not Do

- silently assume business expenses are zero
- silently apply every receipt as a deduction
- pretend unsupported forms are handled
- hide uncertainty about source quality or missing PDFs
